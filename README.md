# Investigating the Effect of Pulling Velocity on the Formation of Tape Stripes

## Introduction

Peeling adhesive tape in daily life often leaves periodic white stripes on the surface. This project aims to investigate the physical relationship between stripe morphology and peeling force during the stick-slip phenomenon.

Using an Arduino-controlled stepper motor and a force sensor, we built a three-pulley traction system to adjust the pulling velocity precisely. We analyze the force-time ($F-t$) curves across three distinct velocity regimes (continuous white, oscillatory, and continuous transparent) to determine the critical peeling forces. Furthermore, we investigate how the stripe spacing ($\lambda$) and the features of white/transparent bands in the oscillation zone vary with the pulling velocity.
